# Abacus AI Engineering Internship

Welcome to my repository for the Abacus AI Engineering Internship.

This repository contains my learning progress, assignments, and projects completed during the 6-week internship. The internship focuses on developing practical AI Engineering skills, starting from Python foundations and gradually moving towards Machine Learning, Deep Learning, Generative AI, and LLM-based application development.

The purpose of this repository is to document my learning journey and showcase the projects developed throughout the internship.

---

## Internship Roadmap

### Week 1: AI Engineering Introduction + Python Foundation

During the first week, the focus is on building a foundation in AI Engineering concepts and strengthening Python programming skills.

Topics covered:

* Introduction to AI and Generative AI
* AI Engineer role and responsibilities
* AI product development lifecycle
* Python fundamentals
* Data structures and functions
* File handling and JSON processing
* Object-Oriented Programming (OOP)
* Error handling and package management
* Working with APIs and structured project development

**Project: AI Document Processing System**

The project focuses on:

* Reading and processing documents
* Extracting and storing information
* Applying OOP principles
* Handling errors
* Creating a structured Python project setup

---

### Week 2: Machine Learning + Deep Learning Fundamentals

The second week focuses on understanding the fundamentals of Machine Learning and Deep Learning.

Topics covered:

* Python for Machine Learning
* NumPy and Pandas
* Dataset processing and cleaning
* Data visualization
* Statistics and probability basics
* Machine Learning workflow
* Supervised and unsupervised learning
* Model training and evaluation
* Introduction to neural networks
* Deep Learning fundamentals

**Project: ML Prediction System + Basic Neural Network**

The project focuses on:

* Processing datasets
* Training Machine Learning models
* Building a basic neural network
* Evaluating model performance
* Generating predictions

---

### Week 3: Generative AI + LLM Application Development

The third week focuses on building an understanding of modern Generative AI applications.

Topics covered:

* Generative AI fundamentals
* Large Language Models (LLMs)
* Transformers and tokenization
* Prompt engineering
* Working with LLM APIs
* Embeddings and vector databases
* Retrieval Augmented Generation (RAG)
* Introduction to AI agents

**Project: AI Knowledge Assistant**

The project focuses on:

* Document upload and processing
* Creating embeddings
* Vector database storage
* Question answering
* AI-generated responses
* Basic AI agent workflows
